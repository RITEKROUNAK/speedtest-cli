# SpeedTest CLI

## Usage

For Using this Cli , curl need to be installed first, then paste the command below in the terminal and execute it.


```

  curl -s https://raw.githubusercontent.com/RITEKROUNAK/speedtest-cli/main/speedtest.py | python -

```
